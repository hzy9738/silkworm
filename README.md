## xxui component library
## 使用 React+typescript 组件库


### 安装

~~~javascript
npm install silkworm --save
~~~

### 使用

~~~javascript
// 加载样式
import 'xxui/dist/index.css'
// 引入组件
import { Button } from 'silkworm'
~~~

### 一些本地开发命令

~~~bash
//启动本地环境
npm run stroybook

//build可发布静态文件
npm run build

//发布到 npm
npm run publish
~~~
